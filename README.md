# Redes Neuronales y Aprendizaje Automático

Este repositorio centraliza mis implementaciones, estudios y proyectos relacionados con la Inteligencia Artificial, desde fundamentos matemáticos en MATLAB hasta aplicaciones avanzadas.

## 📂 Estructura del Repositorio

* **`Fundamentos/`**: Implementaciones base de algoritmos clásicos (Regresión, Optimización).
* **`redes_neuronales_base/`**: Algoritmos de redes neuronales desarrollados desde cero ("from scratch") sin el uso de librerías de alto nivel.
* **`proyectos/`**: Aplicaciones prácticas y casos de estudio.

## 🚀 Proyecto Destacado
Actualmente, el repositorio cuenta con una implementación completa de una **Red Neuronal por Método Patrón (SGD)** en MATLAB, ideal para entender el flujo de señales (Forward y Backpropagation) en hardware embebido.
